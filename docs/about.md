# О проекте

## Назначение

Проект "Учёт успеваемости студентов" разработан для автоматизации работы учебных заведений. Система позволяет отказаться от бумажных журналов и ведомостей, централизованно хранить данные и быстро получать отчёты.

## История версий

### Версия 2.0 (2025)
- Полностью переработан интерфейс
- Добавлена тёмная тема
- Внедрено API для интеграции с LMS

### Версия 1.0 (2024)
- Первый релиз
- Базовый функционал учёта
- Экспорт в Excel
