

# Установка и настройка

## Системные требования

| Компонент | Минимальные | Рекомендуемые |
|-----------|-------------|---------------|
| Процессор | 2 ядра, 1.5 ГГц | 4 ядра, 2.5+ ГГц |
| ОЗУ | 4 ГБ | 8+ ГБ |
| Диск | 500 МБ свободно | 1+ ГБ |
| ОС | Windows 10 / Ubuntu 20.04 / macOS 11 | Windows 11 / Ubuntu 22.04 / macOS 13 |

## Пошаговая установка

### Шаг 1. Загрузка дистрибутива

Скачайте последнюю версию с официального сайта:
https://example.com/download


### Шаг 2. Установка на Windows

1. Запустите `setup.exe` от имени администратора
2. Выберите язык установки (русский/английский)
3. Примите лицензионное соглашение
4. Выберите папку для установки
5. Нажмите "Установить"

### Шаг 3. Установка на Linux

```bash
# Распакуйте архив
tar -xzf student-grades.tar.gz

# Перейдите в папку
cd student-grades

# Запустите установку
sudo ./install.sh
```
### Шаг 4. Подключение к БД

1. Откройте файл `config/database.ini`
2. Укажите параметры подключения:

```ini
[postgresql]
host = localhost
port = 5432
database = grades_db
user = postgres
password = ваш_пароль
```
